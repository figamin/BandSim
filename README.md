# BandSim
Note rhythm game made with PyGame (old)
## Gameplay:
Two modes:
- Main mode with preset notes
- Random mode with random notes
Note range from D4 to E5, with whole, half, and quarter notes
## Requires:
- pygame
- numpy (for sine wave generation)
